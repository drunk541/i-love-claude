# Railway Linux VPS

یه محیط Ubuntu 22.04 روی Railway

## ابزارهای نصب شده
- curl, wget, git
- vim, nano, htop
- screen, tmux
- python3, pip
- nodejs, npm

## نحوه استفاده

1. این repo رو fork کن
2. برو به [railway.app](https://railway.app)
3. New Project → Deploy from GitHub repo
4. بعد از deploy، از Railway CLI یا داشبورد وارد شو

## دسترسی به shell

از داشبورد Railway:
Settings → Deploy → Open Shell
